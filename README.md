# Checkers
## Checkers project made by @dchoptiany and @woycik

*Game variants*
- Polish Checkers
- Russian Checkers
- English Checkers


*Used technology*
- JavaFX
- Maven
- JUnit
## To  run server
`mvn clean javafx:run -Pserver`


## To run client
`mvn clean javafx:run -Pclient`
